**Важная деталь про присваивание:**

## 2.2. Типовой шаблон: заполнить категориальные «Unknown», числовые — средним

```python
for col in df_train.columns:
    if df_train[col].isna().any() or df_test[col].isna().any():
        if df_train[col].dtype == 'object':
            df_train[col] = df_train[col].fillna('Unknown')
            df_test[col] = df_test[col].fillna('Unknown')
        else:
            # Среднее лучше считать по train, чтобы не было утечки!
            mean_val = df_train[col].mean()
            df_train[col] = df_train[col].fillna(mean_val)
            df_test[col] = df_test[col].fillna(mean_val)
```

**Под задачу про кошек** (Task C-2 из примера): нужно посчитать число новых уникальных значений для заполнения. Логика:
- Категориальный столбец с NaN → 1 новая категория ('Unknown') на столбец.
- Числовой столбец с NaN → 1 «среднее» на столбец.
- Если в данном столбце нет NaN → не добавляем ничего.

```python
n_fills = 0
for col in df_train.columns:
    if df_train[col].isna().any() or df_test[col].isna().any():
        n_fills += 1
        # затем сам fillna
print(n_fills)
```

**Тонкость:** один и тот же столбец, заполненный одинаковым значением в train и test, добавляет одну «новую категорию/среднее». Если в задаче явно говорится «одинаково в train и test» — так и считай. Если разные значения (например, среднее по train и test разное), то это уже 2 разных значения.

### **Очень важная деталь — interpolation в quantile**

По умолчанию `quantile` использует **линейную** интерполяцию (между двумя соседними значениями). Можно изменить:

```python
df['col'].quantile(0.25, interpolation='linear')   # по умолчанию
df['col'].quantile(0.25, interpolation='lower')    # ← это в задаче про кошек!
df['col'].quantile(0.25, interpolation='higher')
df['col'].quantile(0.25, interpolation='midpoint')
df['col'].quantile(0.25, interpolation='nearest')
```

В задаче про кошек: «используйте интерполяцию меньшим значением (lower interpolation)» — это:
```python
df['col'].quantile(0.25, interpolation='lower')
df['col'].quantile(0.75, interpolation='lower')
```

**IQR (межквартильный размах) с lower:**
```python
q1 = df['col'].quantile(0.25, interpolation='lower')
q3 = df['col'].quantile(0.75, interpolation='lower')
iqr = q3 - q1
```

# Сводный чек-лист и подводные камни

## 1. **Округление.** Если задача просит «округлите до сотых» — это `round(x, 2)`.

## 2. **`.sum()` vs `len()`.** На булевых масках одинаково. На обычных столбцах разное:
- `(df['col'] > 0).sum()` — число True.
- `len(df[df['col'] > 0])` — то же самое.
- `df['col'].sum()` — сумма значений.

## 3. **Скобки в фильтрах:** обязательны при &, |.
```python
df[(df['a'] > 0) & (df['b'] < 5)]   # ✓
df[df['a'] > 0 & df['b'] < 5]        # ✗ ошибка
```

## 4. **`.mean()` на булевой Series** = доля True.

## 5. **`.mode()` возвращает Series**, бери `[0]`.

## 6. **`fillna` без присваивания не меняет столбец.**

## 7. **`quantile` по умолчанию — линейная интерполяция.** В задаче читай требование!
   - `lower` — меньшее.
   - `higher` — большее.
   - `linear` — между.

## 8. **`.corr()` по умолчанию игнорирует NaN попарно**, но не работает с нечисловыми столбцами.

## 9. **`groupby` по нескольким столбцам** даёт MultiIndex; используй `.reset_index()`.

## 10. **`value_counts` сортирует по убыванию** автоматически и **по умолчанию игнорирует NaN** — добавляй `dropna=False`.

## 11. **Категориальные значения** — это столбцы с dtype `object`. Числовые — `int64`, `float64`. Проверка: `df.dtypes`, `df.select_dtypes(include='number')`.

## 12. **Утечка данных при заполнении пропусков:** среднее/медиану для test всегда считай **по train**, не по test или по объединению.

## 13. **Бинарный столбец из условия:**
```python
df['Pass'] = (cond).astype(int)
```

## 14. **`isin` для проверки принадлежности множеству:**
```python
df[df['col'].isin(['a', 'b'])]
```

## 15. **`between` для диапазона:**
```python
df[df['col'].between(10, 20)]   # включает границы
```

# Шпаргалка «вопрос → код»

| Вопрос | Код |
|---|---|
| Сколько строк? | `len(df)` или `df.shape[0]` |
| Сколько столбцов? | `df.shape[1]` |
| Сколько пропусков в столбце? | `df['col'].isna().sum()` |
| Сколько уникальных в столбце? | `df['col'].nunique()` |
| Какие категории и сколько раз? | `df['col'].value_counts()` |
| Среднее по группе? | `df.groupby('g')['col'].mean()` |
| Сколько объектов по условию? | `(cond).sum()` |
| Доля объектов по условию? | `(cond).mean()` |
| Корреляция? | `df['a'].corr(df['b'])` |
| Медиана? | `df['col'].median()` |
| 1-й квартиль (lower)? | `df['col'].quantile(0.25, interpolation='lower')` |
| IQR? | `q3 - q1` |
| Заполнить пропуски средним? | `df['col'].fillna(df['col'].mean())` |
| Новый столбец по условию? | `df['new'] = (cond).astype(int)` |
| Топ-5 значений? | `df['col'].nlargest(5)` |
| Сортировка по столбцу? | `df.sort_values('col', ascending=False)` |
| Все ли True? | `(cond).all()` |
| Хотя бы один True? | `(cond).any()` |
| Только числовые? | `df.select_dtypes(include='number')` |
| Только категориальные? | `df.select_dtypes(include='object')` |
| Условный фильтр + одиночное значение | `df.loc[cond, 'col'].mean()` |
